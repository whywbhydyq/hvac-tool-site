# HVAC Tool Site

Static-first home comfort calculator hub for room AC BTU, window AC sizing, portable AC sizing, AC tonnage, dehumidifier pints/day, basement dehumidifier sizing, CFM by ACH, ACH, bathroom fan CFM, garage ventilation, room ventilation, and basic HVAC unit conversions.

The project keeps calculator math in source modules, renders static SEO pages, runs calculations in the browser, supports URL parameter sharing, exports CSV results, and preserves explicit professional boundaries.

## Commands

```bash
npm run check
npm run build
npm test
```

## Boundary

Educational preliminary estimates only. Not Manual J, Manual S, Manual D, local-code compliance, electrical sizing, combustion safety, mold remediation, medical advice, or product installation approval.
