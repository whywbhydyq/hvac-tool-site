# HVAC Tool Site

Next.js App Router home comfort calculator hub for room AC BTU, window AC sizing, portable AC sizing, AC tonnage, dehumidifier pints/day, basement dehumidifier sizing, CFM by ACH, ACH, bathroom fan CFM, garage ventilation, room ventilation, and HVAC unit conversions.

The project follows the development plan in `docs/hvac-tool-site/development-plan.md` and the product requirements in `docs/hvac-tool-site/requirements.md`.

## Stack

- Next.js App Router
- TypeScript
- React client calculators
- Tailwind CSS
- Zod validation schemas
- Vitest calculator/content tests
- Playwright smoke test scaffold
- Static-first pages deployable to Vercel

## Commands

```bash
npm install
npm run check
npm run dev
npm run build
npm run test:e2e
```

## Boundary

Educational preliminary estimates only. Not Manual J, Manual S, Manual D, local-code compliance, electrical sizing, combustion safety, mold remediation, medical advice, or product installation approval.
